====================
Odoo school Hospital
====================

A specialized Odoo module designed for healthcare facilities to manage doctors, patients, and clinical visits efficiently.

Features
========

* **Doctor Profiles**: Detailed tracking of licenses, specialties, and experience levels.
* **Internship Program**: Mentorship management between experienced doctors and interns.
* **Patient Management**: Centralized patient database with medical history, allergy alerts, and blood group tracking.
* **Visit Management**: Advanced scheduling system with automated validation and status tracking (Planned, Completed, Cancelled).
* **Diagnosis Tracking**: History of diagnoses linked to specific visits and patients.
* **Automated Logic**: Includes smart-buttons for quick navigation and automated history logging when a personal doctor is changed.

Technical Requirements
======================

* Odoo 19.0 (Module version 19.0.1.2.0)
* Base Odoo modules

Usage
=====

1. Go to **Hospital** menu.
2. Configure **Doctor Specialties** and **Diseases**.
3. Create **Doctor** profiles (set license and schedules).
4. Register **Patients** and assign personal doctors.
5. Create **Visits** to track appointments and record diagnoses.
